# GlobalReading

Lekka aplikacja webowa do nauki czytania globalnego dla dzieci.

## Funkcje

- Lista słów i obrazków: **auto, dom, panda, słońce**.
- Po kliknięciu słowa pojawia się obrazek i mały podpis.
- Wybór języka interfejsu i słówek: **polski (domyślny), angielski, hiszpański**.
- Brak bazy danych i kont użytkowników (wersja statyczna).

## Uruchomienie

Wystarczy otworzyć plik `index.html` w przeglądarce.

Lub uruchomić prosty serwer lokalny:

```bash
python3 -m http.server 8000
```

Następnie przejść do `http://localhost:8000`.
